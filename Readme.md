# Uma nova forma de se organizar Tarefinhas API 📝

### Criei um pequeno crud com o foco de estudo em dotnet, então essa api foi feita usando Dot net 10, usei o minimals api pra faciliar e o banco não relacional sql que é do proprio dotenet usei para facilitar o cadstramento das tarefas, futuramente pretendo subir um container com essa aplicação;

dotnet run
